## NumberCounter - Java Console Application

Проста Java програма, която брои колко пъти е въведено всяко число от 1 до 10.

### Как да компилирате:
```bash
javac NumberCounter.java
```

### Как да стартирате:
```bash
java NumberCounter
```

### Как работи:
1. Програмата ще поиска да въведете число от 1 до 10
2. Въведете желаното число и натиснете Enter
3. Повторете стъпка 2 колкото пъти желаете
4. Въведете 0 за да излезете и да видите резултатите
5. Програмата ще покаже колко пъти е въведено всяко число
